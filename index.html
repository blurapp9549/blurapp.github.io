How Blur.it works

Blur Text

With BLUR.IT you can select a text block or a paragraph or total block to be blurred from the peering eyes , useful for bluring the API keys or credit card details.You can also blur part of a paragarph very useful for redacting information

Blur Images

With BLUR.IT you can select a particular image to be blured or blur all the images on the website from distracting you.With one click you can blur all the images on the website.

Blur Save

With BLUR.IT you can blur all the elements on a website then click on save to persist the selected blur elements , blur once and find it next time when you open the website. No need to repeat yourself.
